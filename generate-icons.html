Vibeverse Arcade Icon Generator

This tool generates cyberpunk-themed PWA icons for Vibeverse Arcade. Click "Generate Icons" to create all the required sizes, then right-click each icon and save it to the /icons folder with the appropriate filename.

Required filenames: icon-72x72.png, icon-96x96.png, icon-128x128.png, icon-144x144.png, icon-152x152.png, icon-192x192.png, icon-384x384.png, icon-512x512.png
